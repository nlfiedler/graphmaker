GraphMaker: Credits

GraphMaker is principally developed and maintained by Nathan Fiedler, with contributions from various individuals. As of version 4.0, GraphMaker owes a great deal to the NetBeans organization. The NetBeans Platform provides the application framework on which GraphMaker is deployed.
