Output and Logging

Output from GraphMaker is generally displayed using the graphical interface, via dialogs, top components, or the NetBeans Output window. This is the normal means by which the program transmits messages to the user. Output to the console is forbidden since in most cases the console will not be visible to the user.

The use of the java.util.logging.Logger class to report errors or warnings is to be avoided. Instead, use the logging facility provided by the NetBeans Open API (e.g. org.openide.ErrorManager).

Storing Settings

All of the user preferences in GraphMaker are now stored using the mechanism provided by the NetBeans Platform. Using the Java Preferences API is to be avoided as that complicates the situation unnecessarily.
