How to Compile, Run, and Debug GraphMaker

The following is an abbreviated set of steps for building GraphMaker from source code. If you encounter complications at any point, please refer to the detailed steps given in the following sections.

  1. JDK 6.0 must be installed to compile GraphMaker.
  2. NetBeans IDE 5.5 must be installed, as GraphMaker is based on the NetBeans Platform.
  3. Make the build: ant build
  4. Run the application: ant run

If that worked for you, then you are on your way to developing GraphMaker. However, if there was a problem, continue reading the following sections for detailed information.

Installing Required Software

Java Development Kit

GraphMaker is written in the Java language, version 6.0. Visit the Java SE page and download the release appropriate for your system. After installing the JDK, make sure to add the bin directory to your PATH.

Apache Ant

Building GraphMaker requires the use of Apache Ant, a build tool written in Java. In general you should use the latest stable release, but you may find that a slightly older release will work as well. Chances are you will have to set the JAVA_HOME environment variable to point to the JDK installation, and possibly also ANT_HOME, which points to the Ant installation directory.

Alternatively, you can just rely on the Ant version that is bundled with the NetBeans IDE, rather than installing a separate version.

NetBeans IDE

GraphMaker is based on the NetBeans Platform, thus to build GraphMaker you will need to install the NetBeans IDE. Developing GraphMaker within the NetBeans IDE is highly recommended, as it makes many of the mundane tasks much easier.

You may also want to download the NetBeans Platform source code and API documentation, which can be found at the URL given above.

Building GraphMaker

Once the JDK and NetBeans are installed, you are ready to build GraphMaker. Invoke the build target on the build.xml in the root of the source tree. This will compile all of the modules and produce the GraphMaker modules.

ant build

Running GraphMaker

Invoke the run target of the top-level build.xml file. You should soon see the GraphMaker splash screen followed by the main window.

ant run

Debugging GraphMaker

Invoke the debug target of the top-level build.xml file. The application will start in debug mode and the IDE debugger will be connected to it automatically. Note that this target only works inside the NetBeans IDE.

ant debug
