API Documentation

Getting NetBeans API Docs

If you are developing GraphMaker modules, you will most likely want to view the API documentation for NetBeans at some point. Each of the NetBeans modules has its own set of API documentation. The easiest way to get the API docs is to invoke the build-javadoc target of the netbeans-src/nbbuild/build.xml build script, where netbeans-src is found in the NetBeans workspace you created when you compiled GraphMaker. The API documentation can then be found in the netbeans-src/nbbuild/build/javadoc directory.

Generating GraphMaker API Docs

See the compiling page for how to set up your build environment for GraphMaker. Once you can compile the GraphMaker source code successfully, you will be ready to make the API documentation.

To make the API documentation for any module, simply select the javadoc target from the top-level build.xml file. The documentation for each module will be generated into a directory called javadoc under each of the module directories.
