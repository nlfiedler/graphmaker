Getting Help

See Also

The following page describes what to do when you run into a problem that is preventing you from getting work done. Please take a moment and read through this page before sending an email asking for help.

  1. Read the User's Guide in the GraphMaker installation directory, which explains most of the primary application functionality.
  2. Read the help pages on the pertinent topics, as there is often something to learn.
  3. Review the FAQ on the web site (code.google.com/p/graphmaker) to see if this problem has come up before.

There are two avenues for asking questions about GraphMaker. The first is to email the author (nathanfiedler@gmail.com) and the second is to send the question to the graphmaker-user mailing list. When asking for help, be sure to mention the GraphMaker version that you are using.
